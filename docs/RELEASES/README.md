# Release Records

This directory contains standalone PQSend release notes and the release
checklist.

PQSend releases before `v1.0.0` are experimental. A release record must state
the supported package format, compatibility status, implemented backend,
security limitations, and whether an external audit has occurred. A release
label is not a claim of production readiness or post-quantum security.

- [Release checklist](release-checklist.md)
- [v0.1.0-alpha.1](v0.1.0-alpha.1.md)

Release-level changes are summarized in the top-level
[CHANGELOG.md](../../CHANGELOG.md).
