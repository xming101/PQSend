# Package Format Documentation

> [!WARNING]
> PQSend is experimental and unaudited. The current backend is X25519-only and
> not post-quantum-secure. The pre-`v1.0.0` format is unstable.

The sole source of truth for the currently implemented `.pqsend` package byte
format is [`FORMAT.md`](FORMAT.md).

This lowercase filename is retained only so existing documentation links do not
break. Do not maintain a second format specification here.

See the [documentation index](README.md) for the rest of the authoritative
project documentation.
